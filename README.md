# UniTrack
Campus Item Lost & Found System
UniTrack is a collaborative college lost-and-found management platform developed as a team project. The main purpose of this project is to help students recover lost items inside universities or college campuses through a centralized digital system.

Students who lose belongings can submit a lost-item application through the platform. If another student or staff member finds the item, they can upload the item details, image, and location where it was found. This creates a faster and more organized way to reconnect lost items with their owners.

The project was developed as part of academic learning and practical software development experience.

# Project Overview

In colleges and universities, students frequently lose important belongings such as:
- ID cards
- Wallets
- Mobile phones
- Earbuds
- Chargers
- Books
- Bags
- Keys
- Watches
- Documents

Usually, there is no proper centralized system to report or track these lost items. UniTrack solves this problem by providing a digital lost-and-found platform where users can report lost items and upload found-item details.

The system improves communication, reduces confusion, and increases the chances of recovering lost belongings.


# Main Features

## Lost Item Submission
Students can:
- Submit lost-item applications
- Add item descriptions
- Mention date and location
- Provide contact details

## Found Item Upload
If someone finds an item, they can:
- Upload item photos
- Mention where the item was found
- Add additional details
- Help connect the item to its owner

## Image Upload System
The platform supports image uploads for:
- Better identification
- Easier verification
- Faster matching process

## Location Tracking
Users can mention:
- Building name
- Classroom area
- Campus location
- Specific place where item was found
